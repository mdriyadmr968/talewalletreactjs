## Installation



Start with cloning this package on your local machine:

```sh
$ npm i testpacksyied
```



## Usage

### Import package on your project

```sh
$ import all from "testpacksyied";
```
### this all will return 5 components which are
- LoginUi

- Nftimage

- WalletUI

- talewallet

- useHook

##### you can import those components and use this directly on your project.


### Destructure all function from a custom hook name useHook

```sh
$ const { handleTablClick, defaultOpen, fetchList, balance, images } =
    all.useHooks();
```
##### Now you can use those function in your project
